Summary: Latitude vs. X

The purpose of this project is to show a web visualization dashboard website using visualizatons of data from the cities csvfile. This dataset lists cities and their corresponding weather data broken into: Max Temp, Humidity, Cloudiness, and Windspeed.

From this landing page you will be able to click on any of the four categorical visuals to access that page, or you can use the plot dropdown in the navbar. Also in the navbar, you will be able to access a comparison page that will provide all four visualizations on the same page for easy comparison.

Lastly, there is a data tab that will display a table containing data in visualizations.
